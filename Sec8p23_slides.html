Case I: A Normal Population with Known \(\sigma\)

Case I: A Normal Population with Known \(\sigma\)

🧪 Let’s start our study of hypothesis testing for \(\mu\) with the simplest scenario: the population is Normal, and the standard deviation \(\sigma\) is known.

  • Null Hypothesis:

\[ H_0: \mu = \mu_0 \]

  • Test Statistic:

\[ Z = \frac{\overline{X} - \mu_0}{\sigma / \sqrt{n}} \]

Note

Note that \(Z\) has a standard normal distribution when \(H_0\) is true.

Steps in a Hypothesis Test Procedure

  1. Identify the parameter of interest and describe it in the context of the problem situation.

  2. Determine the null value and state the null hypothesis.

  3. State the appropriate alternative hypothesis.

  4. Give the general formula for the test statistic

  5. Compute sample quantities, substitute into the formula, and compute the test statistic value.

  6. Determine the P-value.

  7. Compare the P-value to the significance level \(\alpha\)
    and state your conclusion in the context of the problem.

How to Determine the P-value?

Alternative Hypothesis P-Value Determination
\(H_a: \mu > \mu_0\) Area under the standard normal curve to the right of \(z\)
\(H_a: \mu < \mu_0\) Area under the standard normal curve to the left of \(z\)
\(H_a: \mu \ne \mu_0\) \(2 \cdot\) (area under the standard normal curve to the right of \(|z|\))

Case II: A Normal Population with Unknown \(\sigma\)

Case II: A Normal Population with Unknown \(\sigma\)

🔍 Now we consider the more realistic case where the population is Normal, but the standard deviation \(\sigma\) is not known and must be estimated using \(s\).

Null Hypothesis:

\[ H_0\!: \mu = \mu_0 \]

Test Statistic:

\[ T = \frac{\overline{X} - \mu_0}{s / \sqrt{n}} \]

Note

Note that this statistic follows a \(t\) distribution with \(n - 1\) degrees of freedom.

When to Use the \(t\) Test?

Important

Use the \(t\) distribution when:

  • The population is Normal (or approximately so),
  • The standard deviation \(\sigma\) is unknown,
  • You can estimate \(\sigma\) with the sample standard deviation \(s\).

How to Determine the P-value?

Alternative Hypothesis P-Value Determination
\(H_a: \mu > \mu_0\) Area under the \(t\) curve to the right of \(t\)
\(H_a: \mu < \mu_0\) Area under the \(t\) curve to the left of \(t\)
\(H_a: \mu \ne \mu_0\) \(2 \cdot\) (area under the \(t\) curve to the right of \(|t|\))

Case III: Approximate \(z\)-Test for \(\mu\) (Large \(n\), Unknown \(\sigma\))

Case III: Approximate \(z\)-Test for \(\mu\)

Null Hypothesis:

\[ H_0\!: \mu = \mu_0 \]

Test Statistic:

\[ Z = \frac{\overline{X} - \mu_0}{s / \sqrt{n}} \]

Note

This statistic is approximately standard normal when \(n\) is large (\(n \ge 30\) is a common rule of thumb).

When to Use the Approximate \(z\) Test?

Important

Use this test when:

  • \(\sigma\) is unknown
  • \(n\) is large (Central Limit Theorem applies)
  • No strong skew or outliers in data

How to Determine the P-value?

Alternative Hypothesis P-Value Determination
\(H_a: \mu > \mu_0\) Area under the standard normal curve to the right of \(z\)
\(H_a: \mu < \mu_0\) Area under the standard normal curve to the left of \(z\)
\(H_a: \mu \ne \mu_0\) \(2 \cdot\) (area under the standard normal curve to the right of \(|z|\))

Practical Tip

Note

💡 Though the \(t\) distribution is technically appropriate, for large \(n\) the difference between \(z\) and \(t\) becomes negligible. So we use the standard normal distribution for simplicity.

Type II Error and Power Calculations for Case I

Deriving Type II Error (\(\beta\)) for a Lower-Tailed \(z\) Test with \(\sigma\) Known

Suppose we want to test

  • \(H_0: \mu = \mu_0\) vs \(H_a: \mu < \mu_0\)

  • Assume known \(\sigma\) and normally distributed data

  • Significance level: \(\alpha\)

Then the null hypothesis is rejected when the P-value is less than \(\alpha\). The P-value is calculated from the standardized test statistic

\[ z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}} \]

using the standard normal distribution.

Type II Error for \(H_a: \mu < \mu_0\)

Let \(\mu'\) be a specific value less than \(\mu_0\).

We want: \[ \beta(\mu') = P(\text{Fail to reject } H_0 \mid \mu = \mu') \]

That is: \[ \beta(\mu') = P\left(z > -z_\alpha \mid \mu = \mu'\right) \]

Where: \[ z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}} \]

Substitute and Rearrange

Use the expression for \(z\): \[ \frac{\overline{X} - \mu_0}{\sigma/\sqrt{n}} > -z_\alpha \]

Multiply both sides by \(\sigma/\sqrt{n}\): \[ \overline{X} - \mu_0 > -z_\alpha \cdot \frac{\sigma}{\sqrt{n}} \]

Add \(\mu_0\) to both sides: \[ \overline{X} > \mu_0 - z_\alpha \cdot \frac{\sigma}{\sqrt{n}} \]

Change Distribution Assumption

Now assume \(\mu = \mu'\) (alternative is true): \[ \bar{X} \sim N\left(\mu', \frac{\sigma^2}{n}\right) \]

So we standardize: \[ \beta(\mu') = P\left( \bar{X} > \mu_0 - z_\alpha \cdot \frac{\sigma}{\sqrt{n}} \mid \mu = \mu' \right) \]

Standardize Under \(\mu = \mu'\)

Convert to standard normal: \[ \beta(\mu') = P\left( \frac{\bar{X} - \mu'}{\sigma / \sqrt{n}} > \frac{\mu_0 - z_\alpha \cdot \frac{\sigma}{\sqrt{n}} - \mu'}{\sigma / \sqrt{n}} \right) \]

Simplify: \[ \beta(\mu') = P\left( Z > -z_\alpha + \frac{\mu_0 - \mu'}{\sigma / \sqrt{n}} \right) \]

Final Expression for \(\beta(\mu')\)

Now express \(\beta(\mu')\) in terms of the standard normal cdf: \[ \beta(\mu') = 1 - \Phi\left(-z_\alpha + \frac{\mu_0 - \mu'}{\sigma / \sqrt{n}} \right) \]

R Computation

Use pnorm() substituting the appropriate values.

Solving for Sample Size \(n\)

Suppose we specify:

  • Significance level \(\alpha\)

  • Target power \(1 - \beta\)

  • Alternative value \(\mu'\)

We want: \[ \beta = 1 - \Phi\left(-z_\alpha + \frac{\mu_0 - \mu'}{\sigma / \sqrt{n}} \right) \]

Take the quantile of both sides: \[ z_\beta =-z_\alpha + \frac{\mu_0 - \mu'}{\sigma / \sqrt{n}} \]

Final Sample Size Formula

Solve for \(n\): \[ \sqrt{n} = \frac{\sigma (z_\alpha - z_\beta)}{\mu_0 - \mu'} \quad \Rightarrow \quad n = \left[ \frac{\sigma (z_\alpha - z_\beta)}{\mu_0 - \mu'} \right]^2 \]

This gives the required sample size to achieve specified \(\alpha\) and \(\beta\) for a lower-tailed one-sample \(z\)-test.

Summary

  • There are three main frameworks for testing \(H_0: \mu = \mu_0\)

    • Case I: \(z\)-test (Normal population, known \(\sigma\))
    • Case II: \(t\)-test (Normal population, unknown \(\sigma\))
    • Case III: Approx. \(z\)-test (Large \(n\), unknown \(\sigma\))

  • P-values quantify the evidence against \(H_0\)

  • Lower-tailed, upper-tailed, and two-tailed tests have distinct P-value formulas

  • Type II error and power depend on effect size (\(\mu_0-\mu'\)), \(\sigma\), \(n\), and \(\alpha\)

  • R can compute power or required sample size using power.t.test()
