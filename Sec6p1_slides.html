Point Estimators 📍

Important

Definition. A point estimate of a parameter \(\theta\) is a single number considered a sensible value for \(\theta\), based on a sample.

The point estimator is the rule or function used to generate this value. Since it’s computed from a random sample, it is itself a random variable.

Mean Squared Error (MSE)

Note

Definition. The Mean Squared Error of an estimator \(\hat{\theta}\) is:
\[ \text{MSE}(\hat{\theta}) = E[(\hat{\theta} - \theta)^2] \]

We can decompose the MSE:

\[ \text{MSE} = \underbrace{\text{Var}(\hat{\theta})}_{\text{spread}} + \underbrace{[\text{Bias}(\hat{\theta})]^2}_{\text{squared error of bias}} \]

💬 Why does this decomposition matter when comparing estimators?

Unbiased Estimators 🎯

Important

Definition.
An estimator \(\hat{\theta}\) is unbiased for \(\theta\) if:
\[ E(\hat{\theta}) = \theta \]

If not, the bias is: \(\text{Bias}(\hat{\theta}) =E(\hat{\theta}-\theta)\)

🏆 Hall of Fame of Unbiased Estimators

  • The sample mean \(\bar{X}\) is an unbiased estimator of \(\mu\) from a normal distribution.

  • The sample proportion \(\hat{p} = X/n\) is an unbiased estimator of a binomial probability \(p\).

  • The sample variance \(S^2\) is an unbiased estimator of \(\sigma^2\).

Sample Variance: A Setup for Unbiasedness 🧮

We define the sample variance as: \[ S^2 = \frac{1}{n - 1} \sum_{i=1}^n (X_i - \bar{X})^2 \]

  • This uses \(\bar{X}\) in place of the unknown mean \(\mu\).
  • The denominator is \(n - 1\) instead of \(n\) (Why? we’ll find out)

We’ll show that: \[ E(S^2) = \sigma^2 \] So \(S^2\) is an unbiased estimator of population variance!

🧠 Hint: this involves expanding \((X_i - \bar{X})^2\) and working with expectations.

Estimators with Minimum Variance 🧠

Tip

Definition.
An unbiased estimator with the lowest variance among all unbiased estimators is called the Minimum Variance Unbiased Estimator (MVUE).

Theorem (💎 MVUE for the Mean)

Let \(X_1, \dots, X_n \sim \text{iid}(\mu, \sigma^2)\).
Then \(\hat{\theta} = \bar{X}\) is the MVUE of \(\mu\).

✳️ Minimum variance estimators do more with less!

Standard Error 📏

Note

Definition. The standard error of an estimator \(\hat{\theta}\) is:
\[ \sigma_{\hat{\theta}} = \sqrt{\text{Var}(\hat{\theta})} \]

Often, we replace unknown quantities with estimates → estimated standard error:

\[\hat{\sigma}_{\hat{\theta}}\]

🛠️ Used to construct confidence intervals and assess estimator precision.

Parametric Bootstrap 🔁

Note

Motivation.
What if we want to estimate the standard error of an estimator, but:

  • There’s no closed-form formula for the variance?
  • The sampling distribution is too complex to handle analytically?

➡️ Enter the Bootstrap! It uses the plug-in principle:

  • Our sample data informs a model for the population, from which we simulate new samples.

What is the Parametric Bootstrap? 🔁

Note

Instead of resampling from the data directly, we:

  1. Choose a parametric model for the data (e.g., Normal).
  2. Estimate parameters from the observed data.
  3. Generate simulated samples from the model using the estimated parameters.
  4. Recalculate the estimator on each sample to approximate the sampling distribution.

Visualizing the Bootstrap

This is a parametric bootstrap if we simulate from a model
(e.g., \(\mathcal{N}(\hat{\mu}, \hat{\sigma}^2)\)), and a nonparametric bootstrap if we resample the observed data.

Parametric Bootstrap: Step-by-Step

Suppose we want to estimate the variability of \(\hat{\theta}\),
but don’t have a closed-form for \(\operatorname{Var}(\hat{\theta})\).

➡️ We simulate its sampling distribution.

  1. Estimate parameter(s) \(\hat{\theta}\) from original data.
  2. Generate \(B\) samples from the estimated distribution.
  3. Compute a bootstrap estimate \(\hat{\theta}_b^*\) for each.
  4. Use the spread of the \(\hat{\theta}_b^*\) to approximate the SE: \[ S_{\hat{\theta}} = \sqrt{ \frac{1}{B - 1} \sum_{b=1}^B \left( \hat{\theta}_b^* - \bar{\theta}^* \right)^2 } \]

Where \(\bar{\theta}^* = \frac{1}{B} \sum_{b=1}^B \hat{\theta}_b^*\).

Simulating Bootstrap Standard Error

Estimate the SE of the sample mean using a parametric bootstrap:

set.seed(365)
n <- 30
x <- rnorm(n, mean = 10, sd = 2)  # Original sample (population unknown)

# Step 1: Estimate parameters from the data
mu_hat <- mean(x)
sigma_hat <- sd(x)

# Step 2: Generate B bootstrap samples from N(mu_hat, sigma_hat)
B <- 1000
boot_means <- numeric(B)

for (b in 1:B) {
  x_star <- rnorm(n, mean = mu_hat, sd = sigma_hat)
  boot_means[b] <- mean(x_star)
}

# Step 3: Estimate the bootstrap SE
boot_se <- sd(boot_means)

cat("Parametric Bootstrap SE for sample mean:", round(boot_se, 3), "\n")
Parametric Bootstrap SE for sample mean: 0.35

Key Takeaways

  • Parametric bootstrap assumes a model for the population.
  • We simulate samples using parameter estimates from data.
  • Bootstrap SE reflects the sampling variability of the estimator under that model.
  • Validity depends on how well the model fits the real data!
  • When in doubt about the model, use the non-parametric bootstrap.

Summary 🌟

  • A point estimator is a random variable computed from sample data.
  • MSE = variance + bias\(^2\)
  • Unbiased estimators have \(E(\hat{\theta}) = \theta\)
  • MVUE: most efficient among unbiased estimators
  • Standard error quantifies estimator variability
  • Bootstrap provides a flexible, simulation-based alternative
