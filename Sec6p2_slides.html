📍 Motivation: Estimating Parameters

A central task in statistics: Use data to estimate unknown parameters of a distribution.

  • Given a sample \(X_1,\dots,X_n \sim f(x;\theta_1,\dots,\theta_m)\), how do we estimate the parameters?
  • Two classic techniques:
    • Method of Moments
    • Maximum Likelihood Estimation

🧪 Method of Moments (MoM)

📖 Definition

Let \(X_1,\dots,X_n \sim f(x;\theta_1,\dots,\theta_m)\)
The k-th sample moment is:

\[ M_k = \frac{1}{n} \sum_{i=1}^n X_i^k \]

Match the first \(m\) sample moments with the corresponding population moments:

\[ M_k = E(X^k),\quad\text{for } k = 1,\dots,m \]

➡️ Solve the resulting system for \(\theta_1,\dots,\theta_m\)

💡 Example: MoM for Exponential(\(\lambda\))

We have \(X_1,\dots,X_n \sim \text{Exponential}(\lambda)\), and we know:

  • Population mean: \(E(X) = \frac{1}{\lambda}\)

Moment equation: \[ \frac{1}{n} \sum X_i = \frac{1}{\lambda} \quad \Rightarrow \quad \hat{\lambda}_{\text{MoM}} = \frac{1}{\bar{X}} \]

🧮 Example: MoM for Uniform(0,\(\theta\))

  • Population mean: \(E(X) = \frac{\theta}{2}\)

Set \(\bar{X} = \frac{\theta}{2} \Rightarrow \hat{\theta}_{\text{MoM}} = 2\bar{X}\)

Tip

✅ Simple, but doesn’t always respect sample bounds.

📈 Maximum Likelihood Estimation (MLE)

📖 Definition

Given observed data \(x_1,\dots,x_n\), define the likelihood function:

\[ L(\theta_1,\dots,\theta_m) = f(x_1,\dots,x_n;\theta_1,\dots,\theta_m) \]

MLEs are the parameter values \(\hat{\theta}_1,\dots,\hat{\theta}_m\) that maximize the likelihood.

✏️ Let’s Visualize: Likelihood with Bernoulli

  • Suppose \(X_1, \ldots, X_n \stackrel{iid}{\sim} \text{Bernoulli}(p)\) with observed data \(x_1, \ldots, x_n\) equal to 0 or 1.

  • The likelihood function treats the data as fixed and \(p\) as varying:

\[L(p) = \prod_{i=1}^n p^{x_i}(1-p)^{1-x_i}\]

Sum exponents to get:

\[ L(p) = p^{\sum x_i}(1-p)^{n-\sum x_i} \]

  • \(\sum x_i\) = number of 1s (successes), \(\quad n - \sum x_i\) = number of 0s (failures)

📋 MLE: A Four-Step Template

Step Action
1 Write the likelihood: \(L(\theta) = \prod_{i=1}^n f(x_i;\,\theta)\)
2 Take the log-likelihood: \(\ell(\theta) = \ln L(\theta)\); simplify
3 Differentiate \(\ell(\theta)\); set equal to zero
4 Solve for \(\hat{\theta}\); verify it is a maximum



Tip

Steps 3 and 4 assume the maximum occurs in the interior of the parameter space. We’ll see a case soon where that isn’t true.

💬 Try It! MLE for Bernoulli

Let \(X_1,\dots,X_n \sim \text{Bernoulli}(p)\)

  • Likelihood:
    \[ L(p) = p^{\sum X_i} (1 - p)^{n - \sum X_i} \]
  • Log-likelihood:
    \[ \ell(p) = \sum X_i \ln p + \left(n - \sum X_i \right) \cdot \ln(1 - p) \]

Differentiate and solve for \(p\) that maximizes \(\ell(p)\).

➡️ \(\hat{p} = \bar{X}\)

🧮 Why Use Log-Likelihood?

Logarithms turn products into sums!

Instead of maximizing \(L(\theta)\), we often maximize:

\[ \ell(\theta) = \ln L(\theta) \]

✅ Same location of maximum
✅ Easier to differentiate 
✅ Easier algebra and numeric stability

🧮 MLE for Normal(\(\mu\), \(\theta\)) — Setup

Let \(X_1, \dots, X_n \sim \text{Normal}(\mu, \theta)\), where \(\mu\) is the mean and \(\theta\) is the variance.

  • The density is: \[ f(x; \mu, \theta) = \frac{1}{\sqrt{2\pi\theta}} \exp\left(-\frac{(x - \mu)^2}{2\theta}\right) \]

  • The log-likelihood: \[ \ell(\mu, \theta)=\ln\left[\prod_{i=1}^n f(x_i; \mu, \theta) \right]= \sum_{i=1}^n \ln f(x_i; \mu, \theta) \]

🧮 MLE for Normal(\(\mu\), \(\theta\)) — Log-likelihood

\[ \ell(\mu, \theta) = -\frac{n}{2} \ln(2\pi) - \frac{n}{2} \ln(\theta) - \frac{1}{2\theta} \sum_{i=1}^n (x_i - \mu)^2 \]

To find MLEs, differentiate w.r.t. \(\mu\) and \(\theta\).

🧮 MLE for \(\mu\) (first derivative)

Differentiate \(\ell\) w.r.t. \(\mu\): \[ \frac{\partial \ell}{\partial \mu} = \frac{1}{\theta} \sum_{i=1}^n (x_i - \mu) \]

Set derivative to zero: \[ \sum (x_i - \mu) = 0 \Rightarrow \hat{\mu} = \bar{x} \]

🧮 MLE for \(\theta\) (first derivative)

Differentiate \(\ell\) w.r.t. \(\theta\): \[ \frac{\partial \ell}{\partial \theta} = -\frac{n}{2\theta} + \frac{1}{2\theta^2} \sum_{i=1}^n (x_i - \mu)^2 \]

Set derivative to zero and solve: \[ \hat{\theta} = \frac{1}{n} \sum (x_i - \bar{x})^2 \]

✅ MLEs for Normal(\(\mu\), \(\sigma^2\) )

  • MLE for mean: \[ \hat{\mu} = \bar{X} \]

  • MLE for variance: \[ \hat{\theta} = \hat\sigma^2= \frac{1}{n} \sum (X_i - \bar{X})^2 \]

Tip

Note: The variance estimator differs from the unbiased sample variance, which uses \(n - 1\) in the denominator.

💬 Try It! MLE for Uniform[0,\(\theta\)]

Let \(X_1,\dots,X_n \sim \text{Uniform}[0,\theta]\)

  • Likelihood:
    \[ L(\theta) = \begin{cases} \theta^{-n} & \text{if } \theta \ge \max x_i \\ 0 & \text{otherwise} \end{cases} \]
  • MLE:
    \[ \hat{\theta}_{\text{MLE}} = \max(x_1,\dots,x_n) \]

📘 Properties of MLEs

The Invariance Principle

If \(\hat{\theta}\) is the MLE of \(\theta\),
then \(h(\hat{\theta})\) is the MLE of \(h(\theta)\)

🧠 Examples:

  • If \(\hat{\lambda}\) is the MLE for exponential rate, then \(\hat{\mu} = 1 / \hat{\lambda}\) is MLE for mean
  • If \(s^2\) is the MLE for \(\sigma^2\) for a Normal sample, then \(s\) is the MLE for \(\sigma\).

🔍 Large Sample Behavior of MLEs

📖 Proposition

Under general conditions, as \(n \to \infty\), the MLE:

  • Is consistent:
    \[ \hat{\theta} \xrightarrow{P} \theta \]
  • Is approximately unbiased:
    \[ E(\hat\theta)\approx \theta \]
  • Is exactly or approximately the Minimum Variance Unbiased Estimator (MVUE)

⚠️ Caution

Warning

  • Not every parameter has an MLE!

  • Sometimes the likelihood doesn’t have a maximum in the interior.

  • Sometimes it doesn’t exist!

  • Check the form of the likelihood when possible!

✅ Summary: What Did We Learn About Estimation?

  • MoM gives us a simple, intuitive way to estimate parameters by aligning the sample with theoretical moments—but it doesn’t always yield optimal estimates.

  • MLE takes a probabilistic approach: it finds the parameter values that make the observed data most “plausible.” This method uses the full shape of the distribution and becomes more powerful with larger samples.

  • We saw how MLEs behave in the long run—becoming more precise, often unbiased, and having useful properties like invariance under transformation.
